<h3>ジョージタウンを駆け抜ける。旧正月シーズンの健康的な日曜朝</h3>

ペナンの日曜朝の恒例イベント「Occupy Beach Street (OBS) 5km Fun Run 2026」が本日開催され、国内外から約3,000人のランナーが参加しました。

ジョージタウンの歴史地区をスタート・ゴールとするこのコースには、多くの家族連れや観光客も沿道に詰めかけ、ランナーたちに温かい声援を送りました。旧正月シーズンの華やかな雰囲気の中、国際色豊かな参加者たちが爽やかな汗を流しました。イベントにはチョウ・コン・ヨウ州首席大臣も出席し、自ら旗振り役を務めるとともに、地域コミュニティの結束と健康促進の重要性を語りました。

・**イベント名**: Occupy Beach Street (OBS) 5km Fun Run 2026
・**参加者数**: 約3,000人
・**場所**: ジョージタウン中心部（ビーチ・ストリート周辺）

日曜日の歩行者天国（OBS）を活用したこのイベントは、ペナンの活気あるライフスタイルを象徴する行事として、今年も大きな盛り上がりを見せました。
