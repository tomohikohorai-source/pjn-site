<h3>7月12日にジョージタウンで開催。3キロからハーフマラソンまで3部門を用意</h3>

ペナンの世界遺産地区ジョージタウンを舞台に開催される人気のランニングイベント「Alliance Bank Heritage Run 2026」の参加登録が、最終段階に入りました。

このイベントは2026年7月12日（日）に開催される予定で、参加者の体力や目的に合わせて3km、10km、そしてハーフマラソンにあたる21kmの3つのカテゴリーが用意されています。歴史的な街並みを駆け抜けるユニークな体験ができることから、地元住民だけでなく在住外国人や観光客からも高い人気を集めており、マレーシア政府が推進する「Visit Malaysia 2026」キャンペーンを盛り上げる関連イベントの一つとしても位置づけられています。

・イベント名：Alliance Bank Heritage Run 2026
・開催日：2026年7月12日（日）
・開催場所：ジョージタウン世界遺産地区
・カテゴリー：3km、10km、21km

定員に達し次第、募集は締め切られるため、参加を検討されている方は早めの登録をお勧めします。
